## Features Summary by AI

- [latin.dict.yaml](./latin.dict.yaml)

    - 字母混输：西班牙语、匈牙利语、德语、葡萄牙语、法语、其他
    - 特殊功能：`[?` 自动上屏为 `¿`，`[!` 自动上屏为 `¡`

- 日语输入

    - 平片转换：输入过程中按 `F9` 键
    - 简日转换：输入 `]` 后使用中文双拼选择日语汉字

- 中文输入

    - 双拼映射：小鹤双拼
    - 笔画辅助：输入过程中按 `Tab` 键

- [quad-c.yaml](./quad-c.yaml) - key_bindings

    - 切换输入：输入过程中按 `\` 键
    - 简繁转换：`Ctrl+Shift+J`

Weasel user folder 变更：WeaselSetup.exe 执行 → Select
